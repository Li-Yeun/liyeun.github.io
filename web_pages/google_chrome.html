About Google Chrome


History of Google Chrome

The first version of Google Chrome was planned to release on September 3, 2008. The actual first public release was on September 2, 2008 for windows XP. This first version was officially a beta version, the first public stable release was on December 11, 2008.

Google CEO Eric Schmidt actually opposed the idea of an independent web browser related to Google for six years. He stated "at the time, Google was a small company" and did not want to go through "bruising browser wars". Co-founders Larry Page and Sergey Brin hired several Mozilla Firefox developers to build a prototype of Google Chrome. Schmidt stated "it was so good that it forced to change my mind".

Chrome was assembled from 25 different code libraries from Google and third parties such as Mozilla's Netscape Portable Runtime and SQLite. At first Chrome used the WebKit rendering engine to display web pages, they later forked the WebCore component in 2013 to create their own layout engine named Blink.
Blink only uses WebKit's "WebCore" components. Other Components such as Blink its own multiprocess architecture were used in place of WebKit's native implementation.

Features

Google Chrome features a minimalistic user interface, for example merging the address bar and search bar into the "omnibox". Chrome users can synchronize their bookmarks, history and settings across all devices with the browser installed through their Google Account.
Google Chrome has large number of extensions to enhance the browser’s functionalities for special needs. Starting from simple ad blocking to custom extensions. There are two marketplaces to install extensions on Chrome – one is Chrome Web Store and other is G Suite marketplace. Chrome also has a strong browser performance.

Web standards

Google Chrome its first released passed the Acid1 and Acid2 tests. With Version 4.0, Chrome had passed all aspects of the Acid3 test.
As of May 2011, Chrome has a very good support for JavaScript according to ECMA Internationals's ECMPAScript standards conformance tests. Chrome version 37 scored 10 failed out of 11578 passed. In comparison Firefox 19 scored 193 failed/11752 passed.
In 2011, the Chrome rendering engine passed the official CSS 2.1 test suite by W3C with a score of 89.75% (89.38% out of 99.59% covered) passed CSS 2.1 tests.
